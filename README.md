# 近義詞練習簿

A responsive Traditional Chinese synonym quiz based on *Chinese Synonyms Usage Dictionary* (鄧守信主編).

Live site: https://taiwan-chinese-synonym-quiz.mclaughlin2013matthe.chatgpt.site

## Features

- Topic-based synonym practice
- Immediate answer feedback and usage explanations
- Clickable Taiwan Zhuyin and concise English definitions
- Responsive desktop and mobile layout
- A self-contained offline HTML version in `standalone/chinese-synonym-quiz.html`
